# consuhelp-app
Consuhelp é um aplicativo que oferece assistência e consultoria jurídica para os consumidores em geral, no qual o usuário descreve o problema enfrentado em linguagem natural, e recebe um plano de ação estruturado com o melhor caminho administrativo ou judicial para a resolução do conflito, reduzindo o desgaste e o tempo de resposta.
